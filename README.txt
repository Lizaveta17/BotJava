Бот, помогающий найти рецепт интересуещего блюда.

Про 2 задание: Прикрутка к telegram. Теперь пользователь может запустить бота в telegram и общаться с ним там. Бот может поддерживать введение множества диалогов одновременно с разными пользователями.
Также для работы приложения необходим файл telegram.conf;

Про 3 задание: Поиск блюда не только по названию, но и по ингредиентам. Также нужно будет добавить еще данные, чтобы выбор был больше. Пользователь нажимает "Узнать рецепт", бот предлагает выбор: либо по названию, если так, то все как реализовано раньше, либо по ингредиентам, тут уже предлагает список основных ингредиентов, а потом уже блюда, в которых есть выбранный. + доп: реализация всего этого через кнопки для комфортного пользования.
